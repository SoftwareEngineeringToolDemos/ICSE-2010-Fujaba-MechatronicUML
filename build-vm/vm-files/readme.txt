To launch the tool following please follow following steps:
1. launch eclipse from the folder present at the desktop.
2. Create a workspace for the project.
3. Create a general project by right clicking in "Package Explore" window. New -> Project. Under "New Project" window, General -> Project
4. Right click on the newly created project and select MechatronicUML -> Create a new MechatronicUML model.

We have alreasy create a MechatronicUML model for quick launch of the tool. Eclipse will be launched after the VM is up. It contains a demo folder with the proj for tool created.

Also, required binary files are placed in the desktop inside "binaryFiles" folder.
